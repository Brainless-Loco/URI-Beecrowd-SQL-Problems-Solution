# URI / Beecrowd SQL Problems Solutions (PostgreSQL)

This repository contains solutions to SQL problems from **URI Online Judge (now Beecrowd)**.
All solutions are written using **PostgreSQL** syntax and conventions.

## Repository Structure

Each problem is stored as an individual file using the following naming format:

```
<problem_id>.txt
```

### Example:

```
1001.txt
1002.txt
1501.txt
```

Each file contains the SQL query required to solve the corresponding problem.

## Approach

* Solutions are designed specifically for **PostgreSQL**
* Focus is given to:

  * Query correctness
  * Readability
  * Efficient use of joins, aggregations, and subqueries where necessary
* PostgreSQL-specific features (e.g., `WITH`, `FILTER`, `DISTINCT ON`) may be used when appropriate

## Notes

* These queries may not be directly compatible with other SQL dialects such as MySQL, SQLite, or SQL Server without modification
* Problem statements are not included in this repository — refer to Beecrowd for full descriptions

## How to Use

1. Browse the repository
2. Open the file corresponding to the problem ID
3. Copy the SQL query
4. Run it in a PostgreSQL environment or Beecrowd SQL editor

## Purpose

This repository serves as:

* Practice reference for SQL problem solving
* PostgreSQL learning resource
* Quick lookup for Beecrowd SQL solutions
